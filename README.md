# Romanos
Convert roman number in  normal and normal in romans with tests:



Tabla de equivalencias
M -> 1000
D -> 500
C -> 100
L -> 50
X -> 10
V -> 5
I -> 1
MMMCMXCIX = 3999

Ejemplo:
  1123 ==> MCXXIII
  ||||____ III ------> unidades
  |||_____ XX -------> decenas
  ||______ C --------> centenas
  |_______ M --------> unidades de millar
